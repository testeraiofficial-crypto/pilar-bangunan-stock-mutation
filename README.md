# Sistem Laporan Mutasi Barang - Minimarket Bangunan Pilar

Aplikasi web sederhana untuk menstandarisasi format laporan mutasi barang (WhatsApp & Input Sistem).

## Fitur
1.  **Dual Output Generation:** Membuat format laporan WhatsApp (Stylized) dan Format Sistem (Logic string) secara bersamaan.
2.  **Auto Calculation:** Menghitung sisa stok (Asal) dan penambahan stok (Tujuan) secara otomatis berdasarkan Qty Mutasi.
3.  **Modern UI:** Desain responsif dengan tema Gradient Purple/Blue yang User Friendly.
4.  **One Click Copy:** Fitur salin teks cepat.

## Cara Menggunakan
1.  Isi tanggal, nama petugas, dan verifikator.
2.  Masukkan data barang (Nama, Satuan, Jumlah).
3.  Isi lokasi asal dan stok saat ini (Sistem akan menghitung sisa stok otomatis).
4.  Isi lokasi tujuan.
5.  Klik **"Buat Laporan"**.
6.  Salin hasil dari kolom "Laporan WhatsApp" atau "Laporan Sistem" sesuai kebutuhan.

## Logic Format Sistem
Format yang dihasilkan mengikuti pola:
`(MUTASI)(TGL) (PETUGAS) (VERIFIKATOR) (BARANG) DARI (ASAL) KE (TUJUAN) (QTY)`
